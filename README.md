# ANNI - Chatbot Institucional

Chatbot institucional desarrollado mediante Telegram para automatizar consultas académicas.

---

# Tecnologías utilizadas

- Node.js
- Telegraf
- Telegram Bot API
- PostgreSQL
- React

---

# Instalación

## Clonar repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

---

## Instalar dependencias

```bash
npm install
```

---

## Ejecutar proyecto

```bash
node index.js
```

---
